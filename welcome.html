Getting Started

First time? Follow the quick start steps below to configure your endpoint and start chatting in under 2 minutes.

See steps below

Quick Start

  1. 1

    Open the Chat UI

    Click "Open Chat UI" above or navigate to index.html. You'll see the login screen.

  2. 2

    Log in with any name

    Enter your name and click Start Chatting. The system creates a user profile and memory context for you automatically.

  3. 3

    Configure your API endpoint

    Click the ⚙️ gear icon in the header, then Endpoints. Add your Azure Function URL and optional function key. The default points to localhost:7071.

  4. 4

    Start chatting or run a demo

    Type a message to chat with your AI agent, or click the 🎬 Demo Hub button to load and play a scripted demo. Try typing load the sma4 demo to see the SMA4 Workbench Orchestrator in action.

  5. 5

    Explore the Demo Showcase

    Visit the Demo Showcase to browse all 30+ agent demos with conversation previews. Click "Play in RAPP" on any card to run it in the chat.
