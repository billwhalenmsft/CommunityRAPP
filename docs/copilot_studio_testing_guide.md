## 🔧 Troubleshooting

### Agent Not Responding
1. Refresh the test panel
2. Check if the agent is published
3. Verify topics are enabled

### Topic Not Triggering
1. Check trigger phrases in the topic
2. Add more variations to trigger phrases
3. Test with exact trigger phrase first

### Unexpected Fallback
1. The utterance doesn't match any topic trigger
2. Add the phrase to appropriate topic triggers
3. Consider using a more general topic

### Connection Issues
1. Check your internet connection
2. Verify you're signed in correctly
3. Clear browser cache and retry
