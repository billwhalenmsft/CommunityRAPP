# Transpiled

Generated Copilot Studio artifacts from the agent transpiler.

## What goes here

- Transpiled solution folders (Copilot Studio native format)
- Solution ZIP packages
- Deployment manifests
- PAC CLI output

## Workflow

1. Run transpiler on an agent: `agents/my_agent.py` → transpiled output here
2. Test import in Copilot Studio
3. When validated, the generic pattern goes back to `utils/copilot_studio/templates/`
