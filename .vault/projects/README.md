# Projects

RAPP pipeline project workspaces. One folder per engagement.

## Structure

```
projects/
├── acme-corp/
│   ├── inputs/          # Transcripts, feedback docs
│   ├── outputs/         # Generated agents, demos, reports
│   └── notes.md         # Your working notes
├── contoso-retail/
│   └── ...
```

## Naming

Use kebab-case: `customer-name` or `project-codename`
