Setup

  1. Click the ⚙️ Endpoint Configuration section in the panel
  2. For Power Automate mode: paste your flow HTTP URLs
  3. For Azure Fn mode: enter your function key
  4. For Local mode: ensure func start is running
  5. Click Save Configuration — settings persist in localStorage
